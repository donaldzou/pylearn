# PyLearn

## Setup

Make sure to install node v21.5.0 or above, and also require internet access.

Make sure to install the dependencies:

```bash
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```
